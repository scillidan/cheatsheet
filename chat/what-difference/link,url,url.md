What is the difference between link, URL, URI?

```md
The terms "link," "URL," and "URI" are often used interchangeably, but they have distinct meanings in the context of the web and digital technologies. Here's an overview of each term:

### 1. Link
- **Definition**: A "link" (short for hyperlink) refers to a connection or reference to another resource. In a web context, it typically refers to a clickable element (text or image) on a web page that, when clicked, directs the user to another web page, document, or resource.
- **Usage**: The term is commonly used in everyday language, especially when discussing web pages or online content.

### 2. URL (Uniform Resource Locator)
- **Definition**: A URL is a specific type of URI that provides the means to access a resource on the internet. It includes the protocol (such as HTTP or HTTPS), the domain name, and sometimes additional information such as the path to a specific file or resource (e.g., `/path/to/resource`). For example: `https://www.example.com/path/to/page`.
- **Usage**: URLs are used to reference specific locations on the web. They are the addresses you enter in a web browser to navigate to a particular page.

### 3. URI (Uniform Resource Identifier)
- **Definition**: A URI is a broader term that encompasses both URLs and URNs (Uniform Resource Names). It is a string of characters used to identify a resource either by location (like a URL) or by name (like a URN). A URI can be used to refer to resources on the web as well as other types of resources. For example: `urn:isbn:0451450523` is a URN that identifies a book by its ISBN number, while `https://www.example.com` is a URL.
- **Usage**: The term URI is often used in technical specifications and discussions about resources, and it can include both web addresses (URLs) and other forms of identifiers.

### Summary of Differences
- A **link** is a clickable reference to another resource.
- A **URL** is a specific type of URI that provides a means to access a resource via a network using a specific protocol.
- A **URI** is a generic term that describes a way to identify a resource, which can be done by location (URL) or name (URN).

Understanding these definitions can clarify discussions about web resources and their addresses.
```

```md
“链接”、“URL”和“URI”这三个术语常常可以互换使用，但在网络和数字技术的背景下，它们有着不同的含义。以下是每个术语的概述：

### 1. 链接
- **定义**：链接（超链接的缩写）指的是与另一个资源的连接或引用。在网络上下文中，它通常指的是网页上的一个可点击元素（文本或图像），当点击时，会将用户引导到另一个网页、文档或资源。
- **使用**：这个术语在日常语言中被广泛使用，特别是在讨论网页或在线内容时。

### 2. URL（统一资源定位符）
- **定义**：URL是一种特定类型的URI，提供了访问互联网资源的手段。它包括协议（如HTTP或HTTPS）、域名，有时还包括指向特定文件或资源的附加信息（例如：`/path/to/resource`）。例如：`https://www.example.com/path/to/page`。
- **使用**：URL用于引用网络上的特定位置。它们是在网络浏览器中输入的地址，用于导航到特定网页。

### 3. URI（统一资源标识符）
- **定义**：URI是一个更广泛的术语，包括URL和URN（统一资源名称）。它是一串用来标识资源的字符，可以通过位置（像URL）或名称（像URN）来识别。URI既可以用于引用网络资源，也可以用于其他类型的资源。例如：`urn:isbn:0451450523`是一个通过ISBN编号标识书籍的URN，而`https://www.example.com`则是一个URL。
- **使用**：URI这个术语通常在技术规格和关于资源的讨论中使用，它可以包括网络地址（URL）和其他形式的标识符。

### 差异总结
- **链接**是指向另一个资源的可点击引用。
- **URL**是特定类型的URI，通过特定协议提供访问资源的手段。
- **URI**是一个通用术语，描述了识别资源的方式，可以通过位置（URL）或名称（URN）来实现。

理解这些定义可以澄清关于网络资源及其地址的讨论。
```
