What is the difference between list, table, tabular, sheet?

```md
The terms "list," "table," "tabular," and "sheet" refer to different ways of organizing and presenting information, and they each have unique characteristics:

1. **List**:
   - A list is a simple collection of items or entries, usually presented in a single column. 
   - Items in a list can be numbered, bulleted, or unnumbered.
   - Lists are typically used to display related information in a linear fashion, such as a shopping list, to-do list, or a list of features.

2. **Table**:
   - A table is a structured arrangement of data made up of rows and columns. 
   - Each intersection of a row and a column is called a cell, which can contain data, text, or other information.
   - Tables are often used to display complex data that involves comparisons, such as financial data, statistical information, or schedules.

3. **Tabular**:
   - The term "tabular" refers to data that is organized in the same manner as a table, so it implies a format that resembles a table structure.
   - "Tabular" can describe data that is presented in rows and columns, highlighting the organized relationships between different pieces of information.
   - It is often used in contexts such as “tabular data” or “tabular format.”

4. **Sheet**:
   - A sheet, in many contexts, refers to a single page or section of a larger document or work. 
   - In computing, it is often associated with digital spreadsheet software (like Microsoft Excel, Google Sheets, etc.), where a sheet contains multiple cells organized in rows and columns, similar to a table.
   - A "sheet" can contain multiple tables, lists, and other data types and can also accommodate calculations and data analysis.

### Summary
- **List**: Linear collection of items.
- **Table**: Structured data in rows and columns.
- **Tabular**: Refers to data organized like a table.
- **Sheet**: A page/section in a document, often in the format of a spreadsheet with cells.
```

```md
术语“列表”、“表格”、“表格格式”和“工作表”指的是组织和呈现信息的不同方式，它们各自具有独特的特征：

1. **列表**：
   - 列表是一个简单的项目或条目的集合，通常以单列形式呈现。
   - 列表中的项目可以是编号、项目符号或未编号的。
   - 列表通常用于以线性方式展示相关信息，例如购物清单、待办事项清单或功能列表。

2. **表格**：
   - 表格是由行和列构成的结构化数据排列。
   - 行与列的交点称为单元格，单元格可以包含数据、文本或其他信息。
   - 表格常用于展示涉及比较的复杂数据，例如财务数据、统计信息或时间表。

3. **表格格式**：
   - “表格格式”一词指的是以与表格相同的方式组织的数据，暗示着类似表格结构的格式。
   - “表格格式”可以描述以行和列呈现的数据，强调不同信息之间的有序关系。
   - 它常用于“表格数据”或“表格格式”的上下文中。

4. **工作表**：
   - 在许多上下文中，工作表指的是较大文档或作品中的单一页面或部分。
   - 在计算机中，它通常与数字电子表格软件（如 Microsoft Excel、Google Sheets 等）相关联，其中一个工作表包含多个以行和列组织的单元格，类似于表格。
   - 工作表可以包含多个表格、列表和其他数据类型，也可以进行计算和数据分析。

### 总结
- **列表**：线性项目集合。
- **表格**：以行和列结构化的数据。
- **表格格式**：指的是像表格一样组织的数据。
- **工作表**：文档中的一个页面/部分，通常以含有单元格的电子表格格式呈现。
```
