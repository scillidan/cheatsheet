In strictly, what is a non-GUI?

```md
A non-GUI, or non-graphical user interface, refers to any interface that allows users to interact with a computer system or software application without the use of graphical elements like windows, icons, buttons, or menus. Instead, non-GUI interfaces rely on text-based or command-line interactions. Common examples of non-GUI interfaces include:

1. **Command-Line Interfaces (CLI)**: Users interact with the system by typing commands into a terminal or command prompt. Examples include Bash, PowerShell, and the Windows Command Prompt.

2. **Text-Based User Interfaces (TUI)**: These interfaces are more advanced than a simple CLI and can display multiple fields of information (like forms) using text characters to create a more structured layout. An example is `ncurses`, which provides a way to build text-based interfaces for applications.

3. **Application Programming Interfaces (APIs)**: Non-GUI APIs allow programs to communicate with each other or with services without a graphical component, often using protocols like HTTP or various programming libraries.

4. **Scripting Interfaces**: Many programming languages allow scripts to be run that perform actions without the need for a graphical interface, often automating tasks.

Overall, non-GUI interfaces are typically more efficient for experienced users, as they can execute commands quickly without navigating through graphical menus.
```

```md
非图形用户界面（non-GUI）是指任何允许用户与计算机系统或软件应用程序进行交互的接口，而不使用图形元素如窗口、图标、按钮或菜单。相反，非图形用户界面依赖于基于文本或命令行的交互。非图形用户界面的常见示例包括：

1. **命令行接口 (CLI)**：用户通过在终端或命令提示符中输入命令与系统交互。示例包括 Bash、PowerShell 和 Windows 命令提示符。

2. **基于文本的用户界面 (TUI)**：这些接口比简单的 CLI 更先进，使用文本字符显示多个信息字段（如表单），以创建更结构化的布局。一个例子是 `ncurses`，它提供了一种为应用程序构建基于文本的界面的方法。

3. **应用程序编程接口 (API)**：非图形用户 API 允许程序之间或与服务之间进行通信，而无需图形组件，通常使用 HTTP 协议或各种编程库。

4. **脚本接口**：许多编程语言允许运行脚本来执行操作，而无需图形用户界面，通常用于自动化任务。

总体而言，非图形用户界面对经验丰富的用户来说通常更高效，因为他们可以快速执行命令，而无需通过图形菜单进行导航。
```
