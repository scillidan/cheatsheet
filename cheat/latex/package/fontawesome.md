## Quickstart

```tex
\usepackage{fontawesome6}
```

```tex
Multiple versions of the file icon:
    \faFile~
    \faFile[solid]~
    \faFile[regular]~.\\
Alternative syntax:
    \faIcon{file}~
    \faIcon[solid]{file}~
    \faIcon[regular]{file}~.
```
