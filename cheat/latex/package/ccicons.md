## Summary

```md
`ccicons`, a LaTeX package, provides support for typesetting Creative Commons icons. This project allows authors to easily incorporate recognizable symbols representing various Creative Commons licenses in their documents. Featuring an Adobe Type 1 font and custom LaTeX macros, it simplifies the use of these icons, ensuring proper display in published works under Creative Commons licenses. The package aims to promote the understanding of and compliance with Creative Commons licensing through visual representation.

Tags: `LaTeX`, `Creative Commons`, `icons`, `typesetting`, `font`.
```

```md
`ccicons` 是一个 LaTeX 宏包，提供了排版创作共用（Creative Commons）图标的支持。该项目使作者能够轻松地在文档中加入代表各种创作共用许可证的可识别符号。它包含一个 Adobe Type 1 字体和自定义的 LaTeX 宏，简化了这些图标的使用，确保在采用创作共用许可证发布的作品中正确显示。该宏包旨在通过视觉符号促进对创作共用许可的理解和遵守。

标签：`LaTeX`、`创作共用`、`图标`、`排版`、`字体`。
```
