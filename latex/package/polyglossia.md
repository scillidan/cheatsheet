## Summary

```md
**Project Name**: Polyglossia, an alternative to Babel for XeLaTeX and LuaLaTeX tool. This project is designed to streamline multilingual typesetting in LaTeX documents by allowing users to easily switch between different languages and set proper language-specific conventions. Polyglossia supports over 80 languages, automatically loading hyphenation patterns and setting language tags, improving user experience in multilingual text composition.

In summary, Polyglossia enhances LaTeX's capabilities for documents requiring multiple languages, making it a valuable resource for researchers, academics, and writers engaged in polyglot publishing.

Tags: `XeLaTeX`, `LuaLaTeX`, `multilingual`, `typesetting`, `Babel`
```

```md
**项目名称**：Polyglossia，一个针对XeLaTeX和LuaLaTeX工具的Babel替代品。该项目旨在简化LaTeX文档中的多语言排版，允许用户轻松切换不同语言并设置适当的语言特定约定。Polyglossia支持超过80种语言，自动加载连字符模式并设置语言标签，从而改善多语言文本创作中的用户体验。

总之，Polyglossia增强了LaTeX处理需要多种语言的文档的能力，成为研究人员、学者和从事多语言出版的作家的宝贵资源。
```
