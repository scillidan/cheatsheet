## Summary

```md
### DND-5e-LaTeX-Template
`DND-5e-LaTeX-Template` is a LaTeX package specifically designed for typesetting materials related to the fifth edition of Dungeons & Dragons (D&D). It provides a structured way to create character sheets, adventure modules, and other game-related documents with a visual style that closely mirrors that of the official source books. This makes it a valuable tool for both players and Dungeon Masters looking to enhance their game materials.

Tags: `LaTeX`, `Dungeons & Dragons`, `template`, `game document`, `typesetting`
```

```md
### DND-5e-LaTeX-模板
`DND-5e-LaTeX-模板` 是一个专门为排版与第五版《龙与地下城》（D&D）相关材料而设计的 LaTeX 包。它提供了一种结构化的方法来创建角色表、冒险模块和其他与游戏相关的文档，其视觉风格与官方来源书籍非常相似。这使得它成为一个有价值的工具，适合希望增强游戏材料的玩家和地下城主。

标签: `LaTeX`, `龙与地下城`, `模板`, `游戏文档`, `排版`
```
